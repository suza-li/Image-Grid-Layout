# Image Grid Layout
Create a grid with HTML and CSS. The primary focus of this project is to learn and practice the CSS Grid layout technique, which is essential for creating responsive and flexible web layouts.

# Roadmap.sh Link
https://roadmap.sh/projects/image-grid

# Objective of Project
The goal of this project is to gain hands-on experience with CSS Grid, allowing creation of complex layouts with ease.

# Challenges
The challenge of this project was learning more about the grid.
